# 概要

旅行のスケジュールを立てるアプリです。

# 使用した技術スタック

- Swift
- UIKit（Storyboard とコードベースどちらも）
- RxSwift、RxCoCoa
- FSCalander（カレンダーライブライ）
- TimelineTableViewCell（タイムラインライブライ）
- MVC モデル
